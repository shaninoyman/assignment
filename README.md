# Tweetie

## How to Run

1. Mongo
   1. Install local MongoDB server
   2. Make sure it's listening on port XXXX, and no credentials are needed for authentication
2. Install dependencies
   1. `npm i`
3. Run the node server:
   1. `npm run devStart`
